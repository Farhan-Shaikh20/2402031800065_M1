loops in js

loops are used to repeat a block of code until a specified condition is met.

for loop

the for loop is used to run a block of code a certain number of times.


        for (initialization; condition; increment) {
            // code to be executed
        }

while loop

the while loop is used to run a block of code as long as a specified condition is true.


        while (condition) {
            // code to be executed
        }

do...while loop

the do...while loop is similar to the while loop, but it will always execute the block of code at least once, even if the condition is false.


        do {
            // code to be executed
        } while (condition);
